# 🛠️ NodeJS Core Fundamentals & Engineering Path

Welcome to my backend engineering foundation repository. This project documents my transition from a beginner to a developer capable of handling core Node.js modules and custom logic architecture.

## 📂 Repository Structure

### 1. 📈 Learning-Path-Index
- *Objective:* Chronological growth from index1.js to index14.js.
- *Key Concepts:* Evolution of asynchronous logic, basic server setups, and script sequencing.
- *Tools:* Includes a centralized package.json to manage dependencies used during the learning phase.

### 2. ⚙️ Node-System-Modules
This folder demonstrates my expertise in Node's built-in capabilities without external frameworks.
- *HTTP Server:* Manual server creation and request handling (httpModuleWatch.js).
- *File System (FS):* Reading/Writing data and managing server logs (form-practice-one.js).
- *Network & Global:* Fetching data via HTTPS and utilizing global variables like __dirname.

### 3. 🧩 Custom-Logic-Modules
- *Focus:* Clean code and modularity.
- *Content:* Reusable modules for mathematical operations and core JavaScript fundamentals (jsfunda.js).

## 💡 Technical Highlights

- *Form Parsing:* Implemented manual data parsing from HTTP POST requests.
