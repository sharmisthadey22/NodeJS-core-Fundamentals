# NodeJS Core Fundamentals 🚀

This repository is a collection of my foundational learning in Node.js. It documents my journey from understanding basic modules to handling complex file systems and event-driven logic.

### 🛠 Tech & Concepts Covered:
- *Core Modules:* fs, os, path, and events.
- *Async Patterns:* Callbacks and basic asynchronous flows.
- *Custom Modules:* Creating and exporting reusable logic.
- *Progressive Learning:* A series of scripts (index1-index14) showcasing a step-by-step evolution in backend development.
